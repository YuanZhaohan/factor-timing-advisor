# 开平仓规则说明

目前这版择时回测先把规则拆成两步：

第一步先生成事件信号表，格式是：

```text
date / instrument / factor / pattern / signal
```

其中：

```text
signal = 1   表示开仓信号
signal = -1  表示闭仓信号
```

第二步再把不同开仓规则和闭仓规则两两组合，测试每一组规则的交易收益、年化收益、最大回撤和净值表现。

## 开仓规则

### 1. 下方拐点开仓

因子先跌到某个负 sigma 区间以下，例如：

```text
-0.5sigma / -1sigma / -1.5sigma
```

随后确认出现局部谷值并开始向上。

这个规则主要用来测试“低位企稳、均值回复”的买点。

### 2. 上穿阈值开仓

因子从阈值下方上穿到阈值上方时触发。

阈值包括：

```text
-1.5 / -1 / -0.5 / 0 / 0.5 / 1 / 1.5 sigma
```

负阈值上穿偏“低位修复”，0 轴上穿偏“状态转正”，正阈值上穿偏“强势突破”。

### 3. 连续上升开仓

因子连续 `3 / 5 / 10` 天环比上升时触发。

只在刚满足连续上升的第一天触发，避免一段趋势里每天重复打信号。

这个规则主要用来测试因子短期动量改善。

### 4. 斜率转正开仓

因子当前值相对 `3 / 5 / 10` 天前的差分从非正转为正。

这个规则比“连续上升”更宽松，用来捕捉因子从下行切换到上行。

### 5. 上穿均线开仓

因子上穿自身 `20日 / 60日` 均线时触发。

这个规则用因子自己的趋势线判断状态是否改善。

### 6. N日累计上升开仓

因子在 `3 / 5 / 10` 日内累计上升超过：

```text
0.5 / 1 / 1.5 sigma
```

时触发。因为因子本身已经平滑，不再使用单日大幅上升信号，而是用几日内累计变化来捕捉状态改善。

### 7. N日振幅放大向上开仓

过去 `5 / 10 / 20` 日因子的区间振幅 `max - min` 刚超过：

```text
1 / 1.5 sigma
```

且当前因子位于该窗口上半区时触发。这个规则表示波动放大后向上选择。

### 8. 低位持续后回升开仓

因子此前连续 `5 / 10 / 20` 日低于：

```text
-1 / -1.5 sigma
```

随后重新站上该低位阈值时触发。这个规则用来捕捉低位钝化后的脱离。

### 9. 价格因子底背离开仓

指数价格创 `20日 / 60日` 新低，但因子没有同步创新低，且因子当天回升时触发。

这个规则表示价格还弱，但内部因子已经先行修复。

### 10. 价因子同步上升开仓

指数价格过去 `5日 / 20日` 收益为正，同时因子过去同一窗口变化为正时触发。

这个规则表示价格和因子同步改善，偏右侧趋势确认。

### 11. 价格下跌因子上升开仓

指数价格过去 `5日 / 20日` 收益为负，但因子过去同一窗口变化为正时触发。

这个规则表示价格仍弱，但因子先修复，偏左侧抄底观察。

### 12. 价格突破因子确认开仓

指数价格创 `20日 / 60日` 新高，同时因子高于 0 或过去同一窗口因子变化为正时触发。

这个规则表示价格突破时因子也确认，偏趋势共振。

### 13. 价格新低因子未确认开仓

指数价格创 `20日 / 60日` 新低，但因子没有同步创新低，或因子仍高于自身同窗口均线时触发。

这个规则比严格底背离更宽松，用于识别价格新低但内部指标不再恶化的情况。

### 14. 原始/季线/年线多头共振开仓

只在原始指标字段上触发。

当原始指标、季线、年线同步向上，且满足：

```text
原始指标 > 季线 > 年线
```

时触发。这个规则偏右侧确认。

### 15. 低位均值回复开仓

因子前一日处于过去 `20日 / 60日` 的低分位区域，默认低于过去窗口的 20% 分位，且当日开始回升。

这个规则用来测试“低位反弹”的买点。

### 16. process_signal 买点

复用原来的拐点识别函数。

当 `peak_signal == 1` 时作为开仓事件。这个规则比简单拐点多了 `find_peaks` 和滞后确认逻辑。

## 闭仓规则

### 1. 上方拐点闭仓

因子先升到某个正 sigma 区间以上，例如：

```text
0.5 / 1 / 1.5 sigma
```

随后确认出现局部峰值并开始向下。

这个规则主要用于测试“高位转弱”的卖点。

### 2. 下穿阈值闭仓

因子从阈值上方下穿到阈值下方时触发。

阈值包括：

```text
1.5 / 1 / 0.5 / 0 / -0.5 / -1 / -1.5 sigma
```

正阈值下穿偏“强势减弱”，0 轴下穿偏“状态转负”，负阈值下穿偏“趋势恶化”。

### 3. 连续下降闭仓

因子连续 `3 / 5 / 10` 天环比下降时触发。

只在刚满足连续下降的第一天触发，避免重复信号。

主要用于测试因子短期动量转弱。

### 4. 斜率转负闭仓

因子当前值相对 `3 / 5 / 10` 天前的差分从非负转为负。

用于捕捉因子方向从上行切换到下行。

### 5. 下穿均线闭仓

因子下穿自身 `20日 / 60日` 均线时触发。

主要用于测试因子趋势走弱后的退出。

### 6. N日累计下降闭仓

因子在 `3 / 5 / 10` 日内累计下降超过：

```text
0.5 / 1 / 1.5 sigma
```

时触发。因为因子本身已经平滑，不再使用单日大幅下降信号，而是用几日内累计恶化来捕捉趋势转弱。

### 7. N日振幅放大向下闭仓

过去 `5 / 10 / 20` 日因子的区间振幅 `max - min` 刚超过：

```text
1 / 1.5 sigma
```

且当前因子位于该窗口下半区时触发。这个规则表示波动放大后向下选择。

### 8. 高位持续后回落闭仓

因子此前连续 `5 / 10 / 20` 日高于：

```text
1 / 1.5 sigma
```

随后跌破该高位阈值时触发。这个规则用来捕捉高位钝化后的转弱。

### 9. 价格因子顶背离闭仓

指数价格创 `20日 / 60日` 新高，但因子没有同步创新高，且因子当天回落时触发。

这个规则表示价格仍强，但内部因子已经开始走弱。

### 10. 价因子同步下降闭仓

指数价格过去 `5日 / 20日` 收益为负，同时因子过去同一窗口变化为负时触发。

这个规则表示价格和因子同步转弱，偏风险确认。

### 11. 价格上涨因子下降闭仓

指数价格过去 `5日 / 20日` 收益为正，但因子过去同一窗口变化为负时触发。

这个规则表示价格仍强，但内部因子先走弱，偏逃顶预警。

### 12. 价格破位因子确认闭仓

指数价格创 `20日 / 60日` 新低，同时因子低于 0 或过去同一窗口因子变化为负时触发。

这个规则表示价格破位时因子也确认，偏趋势转弱退出。

### 13. 价格新高因子未确认闭仓

指数价格创 `20日 / 60日` 新高，但因子没有同步创新高，或因子低于自身同窗口均线时触发。

这个规则比严格顶背离更宽松，用于识别价格新高但内部指标不再走强的情况。

### 14. 原始/季线/年线空头共振闭仓

只在原始指标字段上触发。

当原始指标、季线、年线同步向下，且满足：

```text
原始指标 < 季线 < 年线
```

时触发。这个规则偏趋势确认式退出。

### 15. 高位均值回复闭仓

因子前一日处于过去 `20日 / 60日` 的高分位区域，默认高于过去窗口的 80% 分位，且当日开始回落。

用于测试高位回落、均值回复式退出。

### 16. 止损闭仓

持仓后，如果区间收益低于：

```text
-3% / -5% / -8%
```

则触发闭仓。

这个规则不能提前生成静态信号，需要在回测中根据入场价动态判断。

### 17. 止盈闭仓

持仓后，如果区间收益高于：

```text
5% / 8% / 12%
```

则触发闭仓。

同样需要根据入场价动态判断。

### 18. 时间退出

持仓满：

```text
5 / 10 / 20 / 60 个交易日
```

后强制闭仓。

用于测试固定持有期退出效果。

### 15. process_signal 卖点

复用原来的拐点识别函数。

当 `peak_signal == -1` 时作为闭仓事件。

## 关于拐点信号

拐点类信号本身需要右侧确认。比如要确认某天是局部低点，必须至少看到后面一天已经开始反弹，所以信号不会发在拐点当天，而是滞后到确认日。

回测时又统一采用“信号出现后，下一交易日才执行”的口径，因此没有未来信息泄露问题。
